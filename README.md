# DC Motor Modelling and PID Optimization

This repository contains MATLAB and Simulink files used in the project:
“Data-Driven Modeling and Metaheuristic-Based PID Optimization for DC Motor Performance Enhancement”

## Folder Structure
- /simscape-model:  DC motor Simscape implementation
- /narx-training:  MATLAB scripts used for NARX training
- /narx-models:  Saved NARX networks and Simulink setups
- /linear-model-validation:  Identified transfer function validation plots and models
- /ga-optimized-controller:  GA-tuned PID implementation

## Tools Used
- MATLAB R2025A
- Simulink
- System Identification Toolbox
- Neural Network Toolbox
